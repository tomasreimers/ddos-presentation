Section 5
What is a DDOS in Technical Terms? - Part 2: The Crash

As the server becomes flooded with requests it can’t respond to actual requests.

However, why the server can’t respond varies.

There are many types of attacks, including:
Application attacks

focus on overloading the server so that it can’t respond because it is too overloaded.

Network attacks

focus on overflowing the medium with which the server connects to the internet (i.e. the server’s bandwidth), thus preventing any other connections to the server.

So how can you avoid becoming part of the problem?

Try to keep your computer clean from spyware. Download an antivirus and prevent yourself from becoming part of a botnet.
